int main ( )

Main entry point for benchmark result visualization.

Reads benchmark results from results.txt, parses the data, and generates a visualization plot showing performance metrics across different matrix sizes and methods. Saves the plot as plot.png.

Returns
EXIT_SUCCESS on successful execution, EXIT_FAILURE on error.
