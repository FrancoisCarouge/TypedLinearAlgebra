Detailed Description

Unit safe linear algebra with mp-units and Eigen.

Demonstrate a variety of linear algebra operations with mp-units and Eigen. This library composes Eigen as the matrix' linear algebra backend with index typed as mp-units types. This sample explicitly uses double precision floating point numbers. This sample uses Eigen linear algebra as the linear algebra backend. This sample uses mp-units types for the strongly typed units.
